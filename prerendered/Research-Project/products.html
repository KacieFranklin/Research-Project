Below shows the key binds for each tool as well as a small description on what the tools do. Feel free to change them to fit your preferences when using the Drawing Tool!

  • Brush: Shift + B

    Brush Tool

    Can be used to draw.

  • Eraser: Shift + E

    Eraser Tool

    Will erase parts of the drawing on the selected layer.

  • Lasso: Shift + L

    Lasso Tool

    Can be used to select parts of the drawing on a selected layer.

  • Drag: Shift + D

    Drag Tool

    Can be used to draw parts of the drawing around.

  • Cut: Shift + C

    Cut Tool

    Will cut out a part of the drawing after using the lasso tool.
